  These are what I've built with AI:

  这些是我和 AI 一起做的东西：

  for managing information

  [DigestDesk](https://digestdesk.nextbigtoy.com/) · [GitHub](https://github.com/nexteacc/digestdesk)
  Your everyday personal newspaper for Substack, YouTube, RSS and more.
  你的每日个人报纸，聚合 Substack、YouTube、RSS 等内容源。

  [Today Vibe Editor](https://today-vibe-editor.fly.dev/) · [GitHub](https://github.com/nexteacc/today-vibe-editor)
  Generate daily and weekly reports from GitHub commit activities.
  将 GitHub 提交活动生成日报、周报。

  for exploring imagination

  [PixarPen](https://pixarpen.nextbigtoy.com/) · [GitHub](https://github.com/nexteacc/PixPen)
  AI image editor powered by Google Gemini, with object selection, editing and cropping.
  基于 Google Gemini 的 AI 图像编辑器，支持对象选择、图像编辑和裁剪。

  [TypeGenAI](https://typegen.nextbigtoy.com/) · [GitHub](https://github.com/nexteacc/typegen)
  Add styles to articles like image filters.
  为文章添加风格，就像给图片加滤镜。

  [Tab Tab](https://chromewebstore.google.com/detail/tab-tab/acgonidbamlgapkkanhflmooljieclbo) · [GitHub](https://github.com/nexteacc/TabTab)
  Browser extension for managing your tabs.
  浏览器插件，用来管理浏览器 Tab 标签。

  [Mega 4](https://mega4labs.vercel.app/) · [GitHub](https://github.com/nexteacc/Mega4Labs)
  Interviews with four major AI labs.
  四大人工智能实验室采访。

  other experiments

  [QRPing](https://qrping.nextbigtoy.com/)
  QR code review generator for local merchants.
  扫码即生成商家好评，解决不知道写什么的烦恼。

  [BananaFood](https://bananafood.app)
  AI menu translation and food recognition app.
  AI 菜单翻译和食物识别应用。
